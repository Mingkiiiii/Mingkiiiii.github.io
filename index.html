MY STORY

 제가 대학교에 들어가 처음 파이썬이라는 코드를 배우면서 성취감을 느낀 프로젝트가 있습니다.

파이썬을 점점 깊게 배우면서 자율주행에 대해 알게 되었고 로봇에 자율주행코딩을 직접 해보면서 코딩에 점점 재미를 붙이기 시작했습니다. 이것은 저에게 큰 성취감을 느끼게 해주었고, 프로그래밍의 길에 한 걸음씩 나아갈 수 있는 힘이 되었습니다.

  • · · ·

[ 2022년도 대학생 때의 작품 ]

 처음 만들어본 작품이라서 부끄럽지만 파이썬과 라이다센서, UbuntuOS 를 이용해 만든 작품입니다. 이러한 경험은 프로그래머로서의 성장에 정말 많은 도움이 되었고, 다른 프로그래밍 언어나 알고리즘을 배우는 것도 수월하게 할 수 있었습니다.
 개발자의 길은 평생을 트렌드를 쫓아 공부해야 한다고 생각합니다. 대학에 와서 처음 본 코드임에도 이런 적성과 흥미가 있었던 저에게 있어서는 개발자는 천직이라고 생각하며 끊임 없이 공부할 것입니다!

  • PHP

    웹 개발을 하는데 이용하는 툴 입니다. 자주 사용하지않지만 한번씩 사용합니다.

    Flask서버를 이용해 만든 AI Model을 테스트하고 비동기 방식으로 Spring서버에 보내줄 수 있습니다.

  • Git

    commit, push, pull, clone 등 git 명령어를 사용하여 레파지토리에 접근할 수 있습니다.

MYRECIPE 유튜브API기반 레시피 공유 커뮤니티PROJECT

  • 간단 소개

    나만의 레시피를 저장하기 위해 만든 레시피 공유 커뮤니티

  • 진행 기간

    2024.03.01 ~ 2024.03.20

  • 핵심 사용 기술

    • Springframework
    • css
    • OracleDB
    • Typed.js
    • AJAX
    • youtube + chatgpt4 API
    • 반응형웹

  • 구성원

    개인 프로젝트

상세 내용

· 기술 선정 이유

그동안 배웠던 Springframework를 이용해 개발해 보기로 했고 거기에 추가로 PHP도 익숙해져야 하니 PHP로도 개발을 해보기로 했다. DB는 스프링에서 OracleDB, PHP에서 MySQL을 사용하였다. MySQL에 대해서 알아보고 싶었고 여러 DB툴을 이용해보고 싶어 MySQL을 사용했다.

· 프로젝트 과정

Figma로 디자인 -> Javascript로 퍼블리싱 -> 데이터 스키마 정의 -> Spring 서버 api 구축 -> 같이 공부하는 개발자 지인들에게 간단한 QA 받고 버그 수정 -> 관리자페이지 구축 -> 유지보수

· 사용 API 및 툴

API : Chatgpt4 VISION, Youtube tool : STS 3.9.11, Pycharm, sqldeveloper, MySQL, PHP, Javascript, Figma, Bootstrap

· 한계점

짧은 기간에 만들다보니 디테일한 부분이 조금 떨어져 조금씩 시간날 때마다 디테일한 부분을 추가하기로 했다.
